# ☕ BrewHaven - Coffee Shop Website

> A modern and responsive coffee shop website showcasing premium blends, products, and locations.

---

## LIVE DEMO 
👉🏻 https://brewheaven-coffee-website.netlify.app/

## 📌 Features

✨ Beautiful landing page with modern UI  
☕ Coffee product showcase (Ethiopian, Colombian, Espresso)  
🎯 Clean navigation (Home, Products, Locations, Contact)  
🛒 Call-to-action buttons (Order Online, Visit Cafe)  
📱 Fully responsive design  
🎨 Stylish dark theme with gold accents  

---

## 🛠 Tech Stack

- HTML5  
- CSS3  
- JavaScript  
- Font Awesome Icons
